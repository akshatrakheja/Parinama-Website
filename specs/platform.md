# Platform — /platform
Goal: prove Parinama is the only fully-integrated climate platform in India.

Design principle: reuse the homepage design system — Geist type, cream/forest/
gold/sage palette, numbered SectionHeader badges, the 1fr/3fr editorial-grid with
its vertical rule, full-width black section rules, square-corner buttons. Keep the
same fonts and text sizes. But vary the *layout inside each section* so the
presentation suits the content: cards, contrast rows, a capability matrix, stat
callouts, a flywheel. Content below is fixed — only its visual treatment changes.

---

     SVG (forest strokes, gold/sage node accents). Sits in the right column or
     spans full width below the sub text. Reinforces the "flywheel" word.
